# Ruta de estudio — JavaScript puro

## Si eres principiante

No intentes abrir todos los archivos a la vez. Haz esto:

1. lee el `README.md` principal del curso;
2. entra al modulo 01;
3. ejecuta los ejemplos;
4. haz los ejercicios;
5. no avances hasta entender que hace el codigo.

## Prioridad real

Lo mas importante primero es dominar:

- variables;
- condiciones;
- funciones;
- arrays;
- objetos;
- DOM;
- eventos;
- asincronia;
- `fetch`.

## Cuando pasar a otra cosa

No saltes a framework hasta que ya puedas:

- leer un script sin perderte;
- manipular el DOM;
- hacer peticiones HTTP;
- entender errores comunes;
- modularizar codigo pequeno.

## Meta del curso

Que JavaScript deje de sentirse como sintaxis rara y pase a ser una herramienta
real para resolver problemas de interfaz, logica y datos.
