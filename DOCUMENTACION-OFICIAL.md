# Documentacion oficial de JavaScript

JavaScript cambia mucho menos de lo que parece cuando estudias sus bases desde
fuentes serias. Estas referencias son las mas utiles para acompanar el curso.

## Referencias principales

- [MDN JavaScript](https://developer.mozilla.org/en-US/docs/Web/JavaScript)
- [JavaScript reference en MDN](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference)
- [ECMAScript standard](https://ecma-international.org/publications-and-standards/standards/ecma-262/)

## Recursos clave para estudiar

- [Funciones](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide/Functions)
- [Objetos](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide/Working_with_objects)
- [Promesas](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Promise)
- [Async functions](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Statements/async_function)
- [Fetch API](https://developer.mozilla.org/en-US/docs/Web/API/Fetch_API)
- [Manipulacion del DOM](https://developer.mozilla.org/en-US/docs/Web/API/Document_Object_Model)

## Orden recomendado

1. Domina variables, tipos y control de flujo.
2. Practica funciones, arrays y objetos.
3. Pasa al DOM y eventos.
4. Luego estudia promesas, async y modulos.
